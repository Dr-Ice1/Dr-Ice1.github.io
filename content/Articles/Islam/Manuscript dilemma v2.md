# Earliest manuscript evidence contradicts core Islamic teachings

# Argument

P1. The Quran confirms the Gospel which was present in the time of Muhammad.\
P2. The Gospel contradicts the Quran on core teachings.\
P3. If the Gospel was already corrupted, the Quran is false, hence Islam is false.\
P4. If the Gospel was not corrupted yet it contradicts the Quran, Islam is false.\
P5. Law of excluding middle.\
C. Islam is false

# P1

All Muslims will agree that the Gospel was given to Jesus. This premise has two parts which must be agreed upon. Firstly, the Gospel was eventually written down so that people who could read, would be able to. Evidence is below:

# P3

This premise is pretty straight forward. If the Gospel, which was confirmed at the time of Muhammad, was corrupt, then the Quran confirms corrupted scriptures. This means Islam is false.

# P4

Here is where the manuscript evidence will matter most. In this premise, the Gospel would not have been corrupted. But they might argue that the Bible today is not what they had at the time of Muhammad. So for this premise, verses have to be provided which contradict the Quran, using manuscript evidence. These manuscripts have to be either from or before the time of Muhammad. These are shown below:

## John 1:1

There are many manuscripts which show this verse and render it exactly the same, here is a short list of a few[^1]:

- Papyrus 75 (written c. 175 – c. 225 AD)
- Papyrus 66 (c. 200; complete)
- Papyrus 5 (c. 250; extant verses: 23–31, 33–40)
- Papyrus 119 (c. 250; extant verses 21–28, 38–44)
- Papyrus 106 (3rd century)
- Codex Vaticanus (c. 325 – c. 350)
- Codex Sinaiticus (c. 350; complete)
- Papyrus 120 (4th century; extant: verses 25–28, 38–44)
- Codex Bezae (c. 400; extant verses 1–15)
- Codex Washingtonianus (c. 400)
- Codex Alexandrinus (c. 400 – c. 440; almost complete)
- Codex Ephraemi Rescriptus (c. 450; extant verses 4–40)
- Codex Borgianus (5th century; extant verses 24–32 in Greek; 16–23 in Sahidic Coptic)
- Papyrus 55 (6th/7th century; extant verses 31–33, 35–38)
- Papyrus 59 (7th century; extant: verses 26, 28, 48, 51)

However, two will be used from the list, the first is P75 which is the earliest. The second is GA05 which comes later.
