\(\mathcal{L}\)aplace Transformation

Definition

The Laplace transformation of \(f(t)\) becomes \(F(s)\) in this way: \[\mathcal{L}\{f(t)\}(s) = F(s) = \int_0^{\infty} e^{-st}f(t) \, \text{d}t \]

We must know how to find the Laplace transformed of a function using the definition, but we do have a list of transformations for differential equations1.

Characteristics

  • \(\mathcal{L}\{f_1(t)\}(s) = \mathcal{L}\{f_2(t)\}(s) \implies f_1 = f_2\)*
  • \(\mathcal{L}\{c_1f_1(t) + c_2f_2(t)\}(s) = c_1\mathcal{L}\{f_1(t)\}(s) + c_2\mathcal{L}\{f_2(t)\}(s)\)
  • \(\mathcal{L}\{f^{(n)}\}(s) = s^n\mathcal{L}\{f(t)\}(s) − s^{n−1}f(0) − \cdots − sf^{(n−2)}(0) − f^{(n−1)}(0)\)

* This is assuming the function is continuous.

Applications

We can use Laplace transformations to solve differential equations.

Step function

Definition

of the Heaviside function \(u_c\) $$ u_c(t)=\begin{cases} 0, & 0 \leq t < c \\ 1, & c \leq t < \infty \end{cases} $$



Theorem

\(\mathcal{L}\{u_c(t)f(t)\}(s) = e^{−cs}\mathcal{L}\{f(t+c)\}(s)\)

On the formula sheet we can find a slightly different version of the same theorem.

Proof

$$ \begin{align*} \mathcal{L}\{f(t)u_c(t)\}(s)&=\int_0^{\infty}f(t)u_c(t)e^{-st}\, \text{d}t\\ &=\int_0^cf(t)u_c(t)e^{-st}\, \text{d}t+\int_c^{\infty}f(t)u_c(t)e^{-st}\, \text{d}t\;\stackrel{\begin{cases}u=t-c\\du=dt\\t=u+c\end{cases}}{\xlongequal{}}\;\int_0^{\infty}f(u+c)u_c(t)e^{-s(u+c)}\, \text{d}u \\ &= e^{-sc}\int_0^{\infty}f(u+c)e^{-su}\, \text{d}u\stackrel{t=u}{\xlongequal{}}e^{-sc}\int_0^{\infty}f(t+c)e^{-st}\, \text{d}t\\ &=e^{-sc}\mathcal{L}\{f(t+c)\}(s) \end{align*} $$

Observation

$$ \begin{align*} f(t)=\begin{cases} f_1(t)&a\leq t<b,\\ f_2(t)&b\leq t<c,\\ f_3(t)&c\leq t<d,\\ \cdots \end{cases} \end{align*} $$

and we can write \(f\) as following:

\(f(t)=f_1(t)(u_a(t)-u_b(t))+f_2(t)(u_b(t)-u_c(t))+f_3(t)(u_c(t)-u_d(t))+\cdots\)

Periodic functions

Assume \(p(t)\) is a periodic function with period \(T\).

Assume $$ f(t)= \begin{cases} p(t), & 0\leq t<T\\ 0, & otherwise \end{cases} $$

Then

$$ \begin{align*} p(t)&=\sum_{k=0}^{\infty}f(t-kT)u_{kT}(t),\\ \mathcal{L}\{p(t)\}(s)&=F(s)\sum_{k=0}^{\infty}e^{-kTs}\stackrel{s>0}{=}\frac{F(s)}{1-e^{-sT}},\\ \text{where,}\\ F(s)&=\mathcal{L}\{f(t)\}(s) \end{align*} $$

Examples

Example 1:

$$ \begin{align*} \begin{cases} R=1 \;\Omega\\ C=0.5\; \text{F}\\ q(0)=3\\ \end{cases} \end{align*} $$
$$ \begin{align*} i(t)R+\frac{1}{C}q(t)&=V_{\text{ext}}(t)=\begin{cases}0, & 0\leq t<\pi \\ 5\cos(t), & t\geq\pi \end{cases}\\ q^{\prime}(t)R+\frac{1}{C}q(t)&=V_{\text{ext}}(t)\\ q^{\prime}(t)+2q(t)&=5\cos(t)\cdot u_{\pi}(t)\\ sQ(s)-3+2Q(s)&=e^{-\pi s}\cdot\frac{-5s}{s^2+1}\\ Q(s)&=\frac{3}{s+2}-e^{-\pi s}\left(-\frac{2}{s+2}+\frac{2s}{s^2+1}+\frac{1}{s^2+1}\right)\\ q(t)&=3e^{-2t}-u_{\pi}(t)\left(-2e^{-2(t-\pi)}+2\cos{(t-\pi)}+\sin{(t-\pi)}\right) \end{align*} $$



Example 2:

$$ \begin{align*} p(t)&=f(t)+f(t-T)+f(t-2T)+\cdots\\ &=\sum_{k=0}^{\infty} f(t-kT)\cdot u_{kT}(t) P(s)=\sum_{k=0}^{\infty} e^{-kTs} \mathcal{L}\{f(t)\}(s)\\ &=\mathcal{L}\{f(t)\}(s)\cdot \color{blue}{\underbrace{\color{black}{\sum_{k=0}^{\infty} (e^{-Ts})^k}}_{\text{Geometric Series}}}\color{black}{}\stackrel{s>0}{=}\mathcal{L}\{f(t)\}(s)\cdot\frac{1}{1-e^{-sT}}\\ &=\frac{F(s)}{1-e^{-sT}} \end{align*} $$


Dirac \(\delta\)elta function

Definition

The Dirac Delta Function in point \(t=t_0\) by:
  • $$\begin{flalign}\delta(t-t_0)=0,\; t\neq t_0&&\end{flalign}$$
  • $$\begin{flalign}\int_{-\infty}^{\infty}\delta(t-t_0)\, \text{d}t=1 &&\end{flalign}$$


🛈 Note: $$\begin{flalign}\color{black}{}\int_{-\infty}^{\infty}\delta(t-t_0)f(t)\, \text{d}t=f(t_0) &&\end{flalign}$$


Proof for note $$ \begin{flalign*} \delta(t-t_0)&=\lim_{\varepsilon\,\downarrow\,0}\varphi_{\epsilon}(t)&&\\ \int_{-\infty}^{\infty}\delta(t-t_0)g(t)\, \text{d}t&=\lim_{\epsilon\,\downarrow\,0}\int_{-\infty}^{\infty}\phi_{\epsilon}(t)g(t)\, \text{d}t\\ \lim_{\epsilon\,\downarrow\,0}\frac{1}{\epsilon}\int_{t_0}^{t_0+\epsilon}g(t)\, \text{d}t &=\lim_{\epsilon\,\downarrow\,0}\frac{1}{\epsilon}\left(G(t_0+\epsilon)-G(t_0)\right)\\ G^{\prime}(t_0)&=g(t_0) \end{flalign*} $$
