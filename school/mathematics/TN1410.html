\(\mathcal{L}\)aplace Transformation

Definition

The Laplace transformation of \(f(t)\) becomes \(F(s)\) in this way: \[\mathcal{L}\{f(t)\}(s) = F(s) = \int_0^{\infty} e^{-st}f(t) \, \text{d}t \]

We must know how to find the Laplace transformed of a function using the definition, but we do have a list of transformations for differential equations1.

Characteristics

  • \(\mathcal{L}\{f_1(t)\}(s) = \mathcal{L}\{f_2(t)\}(s) \implies f_1 = f_2\)*
  • \(\mathcal{L}\{c_1f_1(t) + c_2f_2(t)\}(s) = c_1\mathcal{L}\{f_1(t)\}(s) + c_2\mathcal{L}\{f_2(t)\}(s)\)
  • \(\mathcal{L}\{f^{(n)}\}(s) = s^n\mathcal{L}\{f(t)\}(s) − s^{n−1}f(0) − \cdots − sf^{(n−2)}(0) − f^{(n−1)}(0)\)

* This is assuming the function is continuous.

Applications

We can use Laplace transformations to solve differential equations.

Step function

Definition:
of the Heaviside function \(u_c\) $$ u_c(t)=\begin{cases} 0, & 0 \leq t < c \\ 1, & c \leq t < \infty \end{cases} $$




Theorem
$$\mathcal{L}\{u_c(t)f(t)\}(s) = e^{−cs}\mathcal{L}\{f(t+c)\}(s)$$


On the formula sheet we can find a slightly different version of the same theorem.

Proof

$$ \begin{align*} \mathcal{L}\{f(t)u_c(t)\}(s)&=\int_0^{\infty}f(t)u_c(t)e^{-st}\, \text{d}t\\ &=\int_0^cf(t)u_c(t)e^{-st}\, \text{d}t+\int_c^{\infty}f(t)u_c(t)e^{-st}\, \text{d}t\;\stackrel{\begin{cases}u=t-c\\du=dt\\t=u+c\end{cases}}{\xlongequal{}}\;\int_0^{\infty}f(u+c)u_c(t)e^{-s(u+c)}\, \text{d}u \\ &= e^{-sc}\int_0^{\infty}f(u+c)e^{-su}\, \text{d}u\stackrel{t=u}{\xlongequal{}}e^{-sc}\int_0^{\infty}f(t+c)e^{-st}\, \text{d}t\\ &=e^{-sc}\mathcal{L}\{f(t+c)\}(s) \end{align*} $$

Observation

$$ \begin{align*} f(t)=\begin{cases} f_1(t)&a\leq t<b,\\ f_2(t)&b\leq t<c,\\ f_3(t)&c\leq t<d,\\ \cdots \end{cases} \end{align*} $$

and we can write \(f\) as following:

$$f(t)=f_1(t)(u_a(t)-u_b(t))+f_2(t)(u_b(t)-u_c(t))+f_3(t)(u_c(t)-u_d(t))+\cdots$$

Periodic functions

Assume \(p(t)\) is a periodic function with period \(T\).

Assume $$ f(t)= \begin{cases} p(t), & 0\leq t<T\\ 0, & otherwise \end{cases} $$

Then

$$ \begin{align*} p(t)&=\sum_{k=0}^{\infty}f(t-kT)u_{kT}(t),\\ \mathcal{L}\{p(t)\}(s)&=F(s)\sum_{k=0}^{\infty}e^{-kTs}\stackrel{s>0}{=}\frac{F(s)}{1-e^{-sT}},\\ \text{where,}\\ F(s)&=\mathcal{L}\{f(t)\}(s) \end{align*} $$

Dirac \(\delta\)elta function

Definition

The Dirac Delta Function in point \(t=t_0\) by:
  • $$\begin{flalign}\delta(t-t_0)=0,\; t\neq t_0&&\end{flalign}$$
  • $$\begin{flalign}\int_{-\infty}^{\infty}\delta(t-t_0)\, \text{d}t=1 &&\end{flalign}$$


🛈 Note: $$\begin{flalign}\color{black}{}\int_{-\infty}^{\infty}\delta(t-t_0)f(t)\, \text{d}t=f(t_0) &&\end{flalign}$$


Proof for note $$ \begin{flalign*} \delta(t-t_0)&=\lim_{\varepsilon\,\downarrow\,0}\varphi_{\epsilon}(t)&&\\ \int_{-\infty}^{\infty}\delta(t-t_0)g(t)\, \text{d}t&=\lim_{\epsilon\,\downarrow\,0}\int_{-\infty}^{\infty}\phi_{\epsilon}(t)g(t)\, \text{d}t\\ \lim_{\epsilon\,\downarrow\,0}\frac{1}{\epsilon}\int_{t_0}^{t_0+\epsilon}g(t)\, \text{d}t &=\lim_{\epsilon\,\downarrow\,0}\frac{1}{\epsilon}\left(G(t_0+\epsilon)-G(t_0)\right)\\ G^{\prime}(t_0)&=g(t_0) \end{flalign*} $$

Convolution integrals

If \(F(s)=\mathcal{L}\{f(t)\}(s)\) and \(G(s)=\mathcal{L}\{g(t)\}(s)\), \(s>a\geq 0\), then: $$ H(s)=F(s)G(s)=\mathcal{L}\{h(t)\}(s),\quad s>a $$ with $$ h(t)=\int_0^t f(t-\tau)g(\tau)\, \text{d}\tau=\int_0^t f(\tau)g(t-\tau)\, \text{d}\tau $$


Definition:

  • \(k\) is called convolution of \(f\) and \(g\).
  • $$\begin{flalign}\int_0^t f(t-\tau)g(\tau)\, \text{d}\tau&&\end{flalign}$$ and $$\begin{flalign}\int_0^t f(\tau)g(t-\tau)\, \text{d}\tau&&\end{flalign}$$ are called the convolution integrals.

Piecewise continuous functions

Definition:
\(f\) is called piecewise continuous on \([\alpha,\beta]\) if:
  • \(f\) is, on finitely many points, continuous on \([\alpha,\beta]\)
  • In every \(x\) in \((\alpha,\beta)\) exists \(f(x-)\) and \(f(x+)\)
  • In \(\alpha\) exists \(f(\alpha-)\) and in \(\beta\) \(f(\beta-)\)

Norm

Definition:
Suppose \(f\in \text{PC}[\alpha,\beta]\), the norm is: $$ ||f||=\sqrt{\int_{\alpha}^{\beta} \overline{f(x)}f(x)\, \text{d}x} $$


Note: For \(c\in\mathbb{C}\), \(||cf||=|c|||f||\)


Definition:
The distance between two functions \(f\) and \(g\) is: $$ ||f-g|| $$


Note: \(f=g\) in \(\text{PC}[\alpha,\beta]\) if: $$||f-g||=0$$


Dot product of functions

Definition:
In \(\text{PC}[\alpha,\beta]\), dot product of \(f\) and \(g\) is:

$$ \langle f,g\rangle = \int_{\alpha}^{\beta}\overline{f(x)}g(x) \, \text{d}x $$


  • \( \langle g, f \rangle = \overline{\langle f, g \rangle}. \)
  • \( \langle f, g + h \rangle = \langle f, g \rangle + \langle f, h \rangle \) and \( \langle f + g, h \rangle = \langle f, h \rangle + \langle g, h \rangle. \)
  • If \( c \in \mathbb{C} \) then \( \langle f, c g \rangle = c \langle f, g \rangle \) and \( \langle c f, g \rangle = \overline{c} \langle f, g \rangle \).
  • \( \langle f, f \rangle = \| f \|^2 \geq 0. \)


Note:
$$ \langle f,f\rangle=\int_{\alpha}^{\beta}\overline{f(x)}f(x) \, \text{d}x=||f||^2 $$


Definition:
We call \(g\) orthogonal to \(f\) if:
  • \(\langle f,g\rangle=0\)
  • Set of functions is called orthogonal set if every function is that set is orthogonal to every other function in the set.


Pythagorean theorem:
If \(g\) is orthogonal to \(f\), then \(||f+g||^2=||f||^2+||g||^2\).
In general: if \(\{f_1,f_2,\ldots,f_n\}\) is an orthogonal set, then: $$ ||f_1+f_2+\ldots+f_n||^2=||f_1||^2+||f_2||^2+\ldots+||f_n||^2 $$
voorbeeld, eigenschappen, als dot prod = 0 dan orthogonaal.

Convergence

Definition:
Suppose \(s_1,s_2,s_3,\ldots\) is a sequence in \(\text{PC}[\alpha,\beta]\).
  • The sequence converges (in norm) to the function \(s\) if \(\lim_{n\to\infty}||s_n-s||=0\).

  • The sequence converges pointwise to the function \(s\) if \(\forall x\in [\alpha,\beta]\) the numbersequence \(s_1(x), s_2(x), s_3(x), \ldots\) converges to \(s(x)\).

  • The sequence converges uniformly to the function \(s\) if \(\forall\epsilon>0\; \exists N\mid\forall n\leq N \wedge\forall x\in [\alpha,\beta]\) then \(|s(x)-s_n(x)|<\epsilon\)


Fourier series

Approaching single function

Theorem:

Suppose \(f,\phi\in\text{PC}[\alpha,\beta]\) and \(||\phi||\neq 0\).
Suppose \(c\in \mathbb{C}\) is such that \(f-c\phi\) is orthogonal to \(\phi\).
Then \(\forall c'\neq c \implies ||f-c'\phi||>||f-c\phi||\).


This results in the following:

Suppose \(f,\phi\in\text{PC}[\alpha,\beta]\) and \(||\phi||\neq 0\).
\(||f-c\phi||\) is minimal exactly when $$c=\frac{\langle\phi,f\rangle}{\langle\phi,\phi\rangle}$$

The \(c\) is called the Fouriercoefficient of \(f\).

Proof

$$ \begin{align*} \phi\perp f-c\phi \Leftrightarrow\\ \langle\phi,f-c\phi\rangle=0\Leftrightarrow\\ \langle\phi,f\rangle-c\langle\phi,\phi\rangle=0\Leftrightarrow\\ c\langle\phi,\phi\rangle=\langle\phi,f\rangle\Leftrightarrow\\ c=\frac{\langle\phi,f\rangle}{\langle\phi,\phi\rangle} \end{align*} $$

Approaching more functions

Now looking at more functions:

Suppose the following in \(\text{PC}[\alpha,\beta]\):
  • a function \(f\) which we want to approach;
  • a set of basic functions \(\{\phi_1,\ldots,\phi_n\}\) to use for the approach.

Q: Which scalars \(c_1,\ldots, c_k\in \mathbb{C}\) is \(||f-(c_1\phi_1+\ldots+c_n\phi_n)||\) minimal?

A: Exaclty when \(\langle\phi_k,f-(c_1\phi_1+\ldots+c_n\phi_n)\rangle=0\quad \forall k=1,\ldots,n\).

If the basic functions are orthogonal, then \(\forall k=1,\ldots,n\): $$ \begin{flalign*} c_k=\frac{\langle\phi_k,f\rangle}{\langle\phi_k,\phi_k\rangle}&& \end{flalign*} $$
Helpful tip:
Uneven function integrated across a symmetric interval = 0


Fourier Series

Definition:
Suppose \(f\in\text{PC}[\alpha,\beta]\).
Suppose \(\{\phi_1,\phi_2,\phi_3,\ldots\}\) is an infinite orthogonal set in \(\text{PC}[\alpha,\beta]\), with \(||\phi_k||\neq 0\;\forall k\).
The Fourier series of \(f\) is: $$ \sum_{k=1}^{\infty}c_k\phi_k $$ with \(c_k=\frac{\langle\phi_k,f\rangle}{\langle\phi_k,\phi_k\rangle} \)

Bessel and Parseval

Theorems:
Suppose \(f\in\text{PC}[\alpha,\beta]\).
Suppose \(\{\phi_1,\phi_2,\phi_3,\ldots\}\) is an infinite orthogonal set in \(\text{PC}[\alpha,\beta]\), with \(||\phi_k||\neq 0\;\forall k\). Suppose \(c_k\) are the Fouriercoefficients of \(f\). The series \(\sum_{k=1}^{\infty}|c_k|^2||\phi_k||^2\) is convergent and results in:
$$\sum_{k=1}^{\infty}|c_k|^2||\phi_k||^2\leq||f||^2 \quad\text{Inequality of Bessel}$$ If the Fourier series of \(f\) converges to \(f\) in the norm, that implies:
$$\sum_{k=1}^{\infty}|c_k|^2||\phi_k||^2=||f||^2 \quad\text{Identity of Parseval}$$
